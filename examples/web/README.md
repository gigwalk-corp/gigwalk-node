Webpack React Example
=====================

Simple web example of how to use the API client in browser.


1. Build the client code: `cd {gigwalk-node directory}`
2. `npm i`
3. `npm run build`
4. `cd ./examples/web`
5. `npm i`
6. `npm run webpack`
7. `npm start`
8. Visit [`http://localhost:9999/`](http://localhost:9999/)
